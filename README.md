## Instrucciones

1. Hacer clic en "fork" de este repositorio (botón superior derecho)
2. Abre tu propio fork en Codespaces (botón "Code" > "Codespaces")
3. Ejecuta en la terminal del entorno Codespace los algoritmos correspondientes.
